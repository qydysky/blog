好用的正则

本文记录了常见的正则式。本文默认/正则/g。

大小写转换

大写转小写
^.*$ -> \L$0

小写转大写
^.*$ -> \U$0

驼峰转换

_转大写
(_)(\w) -> \U$2

大写转_
[A-Z] -> _\L$0
